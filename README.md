# LegacyRegion

LegacyRegion — плагин регионов на блоках для Lumi.

Главная идея плагина — дать серверу максимально кастомизируемые регионы без привязки к одному конкретному блоку. Можно подключать разные типы регионов, задавать им радиус, название и правила разрушения.

## Возможности

- Несколько типов регионов через отдельные файлы.
- Подключение своих блоков региона.
- Включение и отключение каждого типа региона через `enabled`.
- Индивидуальный радиус для каждого блока региона.
- Возможность включать и отключать иконку через `button-icon-enabled`.
- Меню управления регионами через формы.
- Настраиваемые частицы при установке региона и отказе действия.
- Голограммы над блоками регионов.
- SQLite и MySQL хранилище.
- Полная настройка сообщений в отдельном `messages.yml`.
- Переключение формата сообщений между чатом и toast-уведомлениями простым переносом ключа.
- Отдельный `menus.yml` для текста всех кнопок и форм меню.

## Сборка

```powershell
.\gradlew.bat build
```

https://github.com/MEFRREEX/FormConstructor и https://github.com/MEFRREEX/JOOQConnector должны лежать рядом с плагином в папке `plugins`.

## Настройки

- `config.yml` настройка команд, частиц, голограмм и регистрация блоков регионов.
- `messages.yml` хранит сообщения. Ключи можно переносить между `chat-message` и `toast-message`, чтобы выбрать вывод в чат или toast-уведомление.
- `menus.yml` хранит текст кнопок и форм меню.
- `regions/*.yml` описывает типы регионов. `enabled` включает или отключает тип региона.
